    About this chatbot

    A rule-based chatbot. No LLM in the hot path. Knowledge base is a curated JSON file derived from the actual Blindfold documentation (README, FAQ, usage, EXAMPLES, contracts, source code) and refreshed by npm run chatbot -- extract.

    Hard rules

    • Never invent APIs, commands, env vars, or file paths.
    • Never paste a real or fake sk-… value.
    • Never log or echo a secret.
    • Always cite the doc / file that backs the answer.
    • Always prefer the canonical doc over a guess.
